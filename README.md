📌 Overview

This project is an upgraded interactive profile card built using vanilla HTML, CSS, and JavaScript. It improves on a basic UI card by adding avatar upload, smoother interactions, and a more refined modern design.

✨ Features
📸 Avatar upload
✏️ Editable profile content
✨ Smooth animations and transitions
🎨 Clean, modern UI design
📱 Responsive layout
🛠️ Tech Stack
HTML
CSS
JavaScript (Vanilla)
